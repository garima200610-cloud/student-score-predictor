# 🎯 Student Score Predictor

This is a Machine Learning web application that predicts student scores based on study hours.

## 🚀 Features

* Predicts score using Linear Regression
* Interactive web interface using Streamlit
* Data visualization (Hours vs Scores graph)

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit
* Matplotlib

## ▶️ Run Locally

```bash
streamlit run app_ui.py
```

## 📊 Dataset

Simple dataset with:

* Hours studied
* Scores
